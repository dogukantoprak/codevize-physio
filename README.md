# Codevize Studio Physical Therapy & Rehabilitation Center

Portfolio-focused healthcare website concept built with React and Vite.

## Overview

This project turns the default Vite starter into a premium, conversion-oriented physical therapy website for the fictional brand **Codevize Studio Physical Therapy & Rehabilitation Center**.

The experience is positioned to outperform typical clinic sites with:

- a stronger first-screen impression
- clearer service positioning
- faster navigation to consultation CTAs
- richer trust signals and outcome storytelling
- cleaner mobile behavior and visual hierarchy

## Stack

- React 19
- Vite 8
- React Router DOM
- Framer Motion
- React Hook Form
- Zod
- React Helmet Async
- Lucide React
- Custom CSS design system

## Pages

- `/` home page with hero, trust bar, services bento, technology tabs, stats, journey, testimonials, insurance, and final CTA
- `/services` services overview page
- `/services/:slug` dynamic service detail pages
- `/about` clinic story, team, facility, awards, and careers content
- `/contact` premium intake form, locations, alternate contact paths, and FAQ

## Highlights

- sticky responsive navigation with services mega menu
- animated, data-driven home page sections
- reusable SEO component with structured data support
- accessible accordion and modal patterns
- validated consultation form with success state
- floating quick-contact actions for mobile and desktop
- portfolio disclaimer included in the footer
